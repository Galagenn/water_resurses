# Техническое Задание (ТЗ) на разработку ИИ-агента для агросектора

## 1. Общие сведения и цели

- **Назначение продукта**: Разработка ИИ-агента для автоматизированного мониторинга, анализа и прогнозирования сельскохозяйственных показателей на основе спутниковых данных.
- **Конечная цель**: Предоставление специалистам и пользователям интерактивного инструмента для повышения эффективности управления водными ресурсами, посевами и мониторинга отклонений.
- **Исполнители**:
  - Бакберген (Бако): Data Scientist / Backend Developer
  - Гальмжан (Гала): Frontend Developer / UI/UX Specialist

## 2. Задачи для Гальмжана (Frontend & UI/UX)

**Гала отвечает за пользовательский опыт, визуализацию и интерактивность системы.**

| № | Задача | Детализация и Требования |
|---|--------|--------------------------|
| 1 | Разработка интерфейса ИИ-агента с интерактивной визуализацией | Создание основного веб-интерфейса (Dashboard) для работы с ИИ-агентом. Интерфейс должен быть адаптивным и интуитивно понятным. Технологии: React/Vue/Angular или другой современный фреймворк, JS/TS, библиотеки для визуализации (например, D3.js, Leaflet/Mapbox для карт). |
| 2 | Интерактивные дашборды | Разработка визуальных панелей для отображения ключевых агрегированных показателей: статистика по водопользованию, площадь посевов, динамика вегетации по регионам/полям. |
| 3 | Графики и тренды | Реализация интерактивных графиков временных рядов (например, NDVI и других индексов) с возможностью детализации по полю/культуре и сравнения периодов. |
| 4 | Карты прогнозов и отклонений | Разработка геоинформационного модуля для отображения: карты посевов, карты фактических орошаемых земель, визуализации зон отклонений (аномалий) на карте с цветовой индикацией. Карта должна поддерживать масштабирование и выбор слоев. |
| 5 | Интеграция с Backend (API) | Настройка взаимодействия с API Бако для получения обработанных данных, прогнозов и карт в реальном или близком к реальному времени. Обеспечение высокой скорости загрузки визуализаций. |
| 6 | Проведение функционального тестирования | Проведение тестирования пользовательского интерфейса (UI/UX), проверка корректности отображения данных, работоспособности фильтров, навигации и интерактивных элементов. |
| 7 | Организация обучающих сессий | Подготовка материалов (снимков экрана, описаний) и демонстрации по работе с интерфейсом для обучающих сессий. (Координация с Бако по содержанию). |
